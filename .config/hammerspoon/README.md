# Hammerspoon Configuration For QWERTY Keyboard

Config reloading works automatically whenever files in `~/.config/hammerspoon/` change.  

The hyper key is defined as `caps lock`.  
Press this plus:  

Key | Action
---|---
delete | Lock Screen

## Launch Apps
Key | App
---|---
g | Google Chrome
s | Safari
a | Opera
f | Finder
m | Mail
return | System Preferences
z | iTunes
x | Screen Sharing
; | Screenshot.app
e | TextEdit
\ | Skype
] | Messages
c | Visual Studio Code
p | PhpStorm
d | TablePlus
t | iTerm2
v | Windscribe VPN


## Workspace Management
Key | Action
---|---
1 | development
2 | communication


## Window Management
Key | Action
---|---
i | Fit window left
o | Fit window right
8 | Fit window upper left
9 | Fit window upper right
k | Fit window lower left
l | Fit window lower right
/ | To next screen
' | Snap all to grid
(space) | Maximize window
